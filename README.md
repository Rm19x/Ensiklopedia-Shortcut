#  Ensiklopedia Shortcut Keyboard  (by Mr.Rm19)

Pusat dokumentasi navigasi, *shortcut* keyboard rahasia, dan trik produktivitas tinggi untuk berbagai sistem operasi, aplikasi pengembangan, hingga platform media sosial populer. Proyek ini dibangun menggunakan struktur web statis yang ringan, responsif, dan dilengkapi fitur pencarian langsung (*Live Search*) berbasis JavaScript tanpa memuat ulang halaman (*zero-reload*).

##  Struktur & Penjelasan File Proyek

Berikut adalah peta folder dan penjelasan fungsi dari masing-masing file yang ada di dalam repositori ini:

###  File Arsitektur Utama
*   **`index.html`** – Dashboard utama atau menu utama interaktif yang berfungsi sebagai gerbang masuk. Menampilkan grid navigasi visual yang rapi untuk mengakses seluruh kategori shortcut.
*   **`style.css`** – Berkas *stylesheet* global dengan tema *Dark Mode* modern untuk memberikan tampilan visual yang konsisten, scannable, dan nyaman di mata pengguna.
*   **`script.js`** – Logika JavaScript utama yang menyuntikkan fitur *Live Search*. Berfungsi menyaring baris tabel shortcut secara instan saat pengguna mengetik kata kunci.

###  Kategori Sistem Operasi (OS)
*   **`windows.html`** – Koleksi shortcut esensial untuk ekosistem Microsoft Windows (manajemen jendela, akses cepat menu sistem, dan trik navigasi).
*   **`linux.html`** – Panduan shortcut untuk lingkungan desktop Linux (GNOME/KDE) dan navigasi sistem dasar.
*   **`macos.html`** – Panduan navigasi khusus untuk Apple Mac, mempermudah pengguna dalam memahami kombinasi tombol `Cmd ⌘`, `Option ⌥`, dan `Control ⌃`.

###  Kategori Developer & Terminal
*   **`coding.html`** – Daftar lengkap shortcut produktivitas untuk penulisan kode cepat di Visual Studio Code (VS Code).
*   **`terminal-cli.html`** – Shortcut baris perintah (*command line interface*) tingkat lanjut untuk mempercepat kontrol kursor dan manipulasi teks di Bash, Zsh, dan PowerShell.
*   **`git.html`** – Rangkuman perintah cepat dan shortcut manajemen kontrol versi (*version control*) Git untuk alur kerja repositori lokal dan remote.

###  Kategori Produktivitas & Aplikasi Umum
*   **`browser-general.html`** – Kombinasi tombol universal yang valid di berbagai web browser populer seperti Google Chrome, Firefox, Edge, dan Brave.
*   **`youtube.html`** – Shortcut rahasia pemutar video YouTube untuk mengontrol penuh video, audio, dan frame navigasi murni tanpa menggunakan mouse.
*   **`microsoft-office.html`** – Tip pemformatan teks cepat dan kontrol presentasi untuk Microsoft Word dan PowerPoint.
*   **`excel-pro.html`** – Shortcut manipulasi data massal, penguncian rumus absolut, dan manajemen baris/kolom tingkat profesional untuk Microsoft Excel.

### Kategori Desain & Teks Editor
*   **`photoshop.html`** – Shortcut andalan para desainer grafis untuk mengelola layer, memilih tools kuas, dan koreksi warna instan di Adobe Photoshop.
*   **`notepad-plus-plus.html`** – Trik pengeditan teks massal, pencarian ekspresi reguler (Regex), dan mode edit kolom vertikal khusus di Notepad++.

### Modul Lintas Platform & Rahasia
*   **`social-media.html`** – Cara menavigasi feed, memberikan interaksi suka/komentar, dan berpindah halaman secara kilat di platform Twitter/X, Facebook, dan LinkedIn.
*   **`hidden-hacks.html`** – Halaman khusus berisi trik kombinasi tombol darurat, penanganan sistem saat *freeze/crash*, dan pintasan penyelamat komputer yang jarang diketahui publik.

## Fitur Unggulan Proyek

1.  **Live Search Anti-Lemot:** Pengguna bisa mencari pintasan apa saja secara langsung lewat satu kolom input teks di setiap halaman.
2.  **Dark Mode Aesthetics:** Desain gelap yang elegan dengan penekanan tag `<kbd>` yang jelas untuk kenyamanan membaca.
3.  **Struktur Kode Bersih:** HTML semantik, CSS modular, dan JS murni (*Vanilla JS*) tanpa ketergantungan library pihak ketiga (*Zero Dependencies*).
