Daftar Shortcut Penyelamat & Hidden Hacks

Kombinasi Tombol Fungsi / Skenario Penggunaan Nyata
Win + Ctrl + Shift + B Reset Driver VGA (Windows): Penyelamat utama saat PC tiba-tiba lag parah, game freeze, atau layar nge-blank hitam. Shortcut ini memaksa sistem merestart driver kartu grafis tanpa mematikan komputer.
Ctrl + Alt + F1 (atau F2-F6) Membuka TTY Teks (Linux): Jika antarmuka desktop Linux (Ubuntu/Fedora/dll) macet total dan mouse tidak bisa digerakkan, tekan ini untuk masuk ke terminal murni untuk mematikan aplikasi penyebab crash lewat perintah CLI.
Cmd ⌘ + Option ⌥ + Shift ⇧ + Esc (Tahan 3 detik) Force Quit Instan (macOS): Mematikan secara paksa aplikasi terdepan yang sedang aktif tanpa perlu membuka jendela daftar aplikasi terlebih dahulu. Sangat ampun untuk aplikasi yang hang.
Win + V Multi-Clipboard Clipboard (Windows): Fitur bawaan yang jarang diaktifkan orang. Memungkinkan kamu melihat riwayat copy-paste hingga 25 data ke belakang, bukan cuma data terakhir yang di-copy.
Alt + Tarik Mouse Vertikal Multi-Line Editing (VS Code / Notepad++): Memblok teks secara vertikal untuk mengetik atau menghapus kode di puluhan baris sekaligus secara sejajar.
Shift + Klik Kanan pada Folder Open PowerShell Here Advanced (Windows): Memunculkan menu klik kanan tersembunyi untuk membuka terminal langsung di jalur folder tersebut (pada Windows versi lama/tertentu).
Win + X lalu tekan U dua kali Shutdown Tercepat (Windows): Jalan pintas mematikan komputer tanpa menyentuh mouse. Win+X membuka menu sistem, U memilih opsi shut down, dan U kedua langsung mengeksekusinya.
Cmd ⌘ + Shift ⇧ + . (Titik) Tampilkan File Tersembunyi (macOS): Menampilkan atau menyembunyikan kembali file-file sistem bermulaan titik (.htaccess, .DS_Store) di dalam Finder secara instan.
