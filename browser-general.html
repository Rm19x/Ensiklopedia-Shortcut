Daftar Lengkap Shortcut Browser (Windows/Linux)

Kombinasi Tombol Fungsi / Deskripsi Nyata
Ctrl + T New Tab: Membuka tab baru yang bersih secara instan tanpa perlu mengklik ikon tanda tambah (+).
Ctrl + W atau Ctrl + F4 Close Tab: Menutup tab aktif yang sedang kamu lihat saat ini secara instan.
Ctrl + Shift + T Undo Close Tab (Penyelamat): Membuka kembali tab browser yang baru saja tidak sengaja kamu tutup, lengkap dengan riwayat halaman sebelumnya.
Ctrl + N New Window: Membuka jendela browser baru yang terpisah.
Ctrl + Shift + N Incognito Window: Membuka jendela mode penyamaran (Incognito/Private) secara instan (Di Firefox/Edge menggunakan Ctrl + Shift + P).
Ctrl + Tab atau Ctrl + Page Down Next Tab: Berpindah ke tab berikutnya di sebelah kanan.
Ctrl + Shift + Tab atau Ctrl + Page Up Previous Tab: Berpindah ke tab sebelumnya di sebelah kiri.
Ctrl + 1 sampai 8 Jump to Specific Tab: Melompat langsung ke urutan nomor tab tertentu (Misal: Ctrl + 1 untuk tab pertama, Ctrl + 3 untuk tab ketiga).
Ctrl + 9 Jump to Last Tab: Langsung melompat ke tab paling ujung kanan, tidak peduli berapa banyak tab yang sedang terbuka.
Ctrl + L atau F6 atau Alt + D Focus Address Bar: Langsung menyorot kolom URL/alamat web di bagian atas, membuatmu bisa langsung mengetik alamat web baru tanpa klik mouse.
Ctrl + R atau F5 Reload Page: Memuat ulang (refresh) halaman web aktif saat ini.
Ctrl + Shift + R atau Shift + F5 Hard Reload: Memuat ulang halaman web sekaligus membersihkan cache browser khusus untuk halaman tersebut (Sangat penting untuk web developer agar perubahan CSS/JS terbaru langsung terlihat).
Alt + ← atau Backspace Go Back: Kembali ke halaman web yang diakses sebelumnya (History Back).
Alt + → atau Shift + Backspace Go Forward: Maju ke halaman web berikutnya (jika kamu baru saja menekan tombol kembali).
Ctrl + H History Panel: Membuka tab riwayat penjelajahan web yang pernah kamu buka sebelumnya.
Ctrl + J Downloads Panel: Membuka tab daftar unduhan file di browser.
Ctrl + D Add Bookmark: Menyimpan halaman web aktif saat ini ke dalam daftar penanda (*bookmark*).
Ctrl + Shift + O Bookmark Manager: Membuka halaman pengelolaan seluruh bookmark yang tersimpan untuk dirapikan.
Ctrl + Shift + Delete Clear Browsing Data: Jalan pintas langsung menuju menu pembersihan data penjelajahan (hapus cookie, cache, dan history penjelajahan).
F12 atau Ctrl + Shift + I Developer Tools: Membuka panel inspeksi elemen kode, konsol JavaScript, dan network untuk kebutuhan debugging web.
Ctrl + F Find text: Membuka kolom pencarian kata di dalam halaman web aktif untuk menemukan teks tertentu dengan cepat.
