Kombinasi Tombol Fungsi / Deskripsi Nyata
Ctrl + L Delete Line: Menghapus satu baris tempat kursor berada secara instan.
Ctrl + I Split Lines: Memotong baris teks yang terlalu panjang ke kanan secara otomatis berdasarkan batas lebar halaman saat ini.
Ctrl + J Join Lines: Menggabungkan baris teks di bawahnya langsung naik ke baris aktif di atasnya secara rapi.
Ctrl + D Duplicate Line: Menggandakan (duplikat) baris teks aktif saat ini tepat ke satu baris di bawahnya.
Ctrl + Shift + ↑ / ↓ Move Line Up/Down: Menggeser atau memindahkan baris teks aktif ke atas atau ke bawah tanpa proses cut-paste.
Ctrl + Q Toggle Single Line Comment: Mengubah baris teks menjadi komentar kode (comment) atau sebaliknya secara instan.
Alt + Shift + Panah atau Alt + Tarik Mouse Column Mode Selection: Menyeleksi teks secara vertikal berbentuk kotak (blok kolom). Sangat sakral untuk menyisipkan atau menghapus karakter di banyak baris berbeda pada posisi yang sejajar.
Alt + C Column Editor: Membuka panel ajaib untuk menyisipkan teks, simbol, atau nomor urut otomatis (1, 2, 3...) ke seluruh baris yang sedang diseleksi dalam mode kolom.
Ctrl + F Find Menu: Membuka jendela pencarian kata di dalam file aktif.
Ctrl + H Replace Menu: Membuka menu temukan teks dan ganti massal (mendukung mode pencarian teks biasa, Extended, dan Regular Expression / Regex).
Ctrl + Shift + F Find in Files: Mencari kata kunci tertentu di seluruh file dokumen yang berada di dalam suatu folder komputer tanpa perlu membuka filenya satu per satu.
F3 / Shift + F3 Find Next/Previous: Melompat langsung ke kata hasil pencarian berikutnya F3 atau sebelumnya Shift+F3.
Ctrl + F2 Toggle Bookmark: Menandai atau memberikan pin biru (bookmark) pada baris teks tertentu sebagai tanda penting.
F2 Next Bookmark: Melompat cepat antar baris yang sudah ditandai oleh bookmark biru di sepanjang dokumen.
Shift + F2 Previous Bookmark: Mundur ke baris bertanda bookmark sebelumnya.
Ctrl + Shift + U UPPERCASE: Mengubah paksa blok teks terpilih menjadi HURUF KAPITAL SEMUA.
Ctrl + U lowercase: Mengubah paksa blok teks terpilih menjadi huruf kecil semua.
Ctrl + Roda Mouse Zoom In / Out: Memperbesar atau memperkecil ukuran teks tampilan dokumen agar nyaman dibaca.
F11 Full Screen Mode: Mengubah tampilan jendela Notepad++ menjadi layar penuh, menyembunyikan semua menu tab agar kamu bisa fokus membaca teks.
Ctrl + Alt + S Save As: Menyimpan dokumen aktif saat ini sebagai file baru dengan nama atau ekstensi format yang berbeda.
