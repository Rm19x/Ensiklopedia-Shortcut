Kombinasi Tombol Fungsi / Deskripsi Nyata
Ctrl + Tombol Panah Jump to Edge: Melompat langsung ke sel paling ujung (atas, bawah, kiri, atau kanan) yang berisi data tanpa perlu melakukan scroll manual.
Ctrl + Shift + Tombol Panah Select Data Range: Memblok atau menyeleksi seluruh sel berisi data mulai dari posisi kursor hingga ke sel paling ujung secara instan.
Ctrl + A Select All Data: Menyeleksi seluruh tabel aktif yang terhubung dengan data saat ini.
Ctrl + Home Jump to A1: Kembali melompat ke sel pertama A1 di bagian paling pojok kiri atas lembar kerja.
Alt + = (Sama Dengan) AutoSum: Menuliskan rumus penjumlahan otomatis =SUM() untuk semua angka yang ada di atas atau di sebelah kiri sel aktif secara instan.
F4 (Saat menulis rumus) Absolute Reference ($): Mengunci posisi Baris atau Kolom sel dalam rumus menjadi referensi absolut (menambahkan tanda dolar secara otomatis seperti $A$1) agar tidak bergeser saat rumus di-copy ke bawah.
Ctrl + D Fill Down: Menyalin data atau rumus yang ada di sel tepat di atasnya ke dalam sel aktif saat ini.
Ctrl + ; (Koma Titik) Insert Current Date: Memasukkan tanggal hari ini secara otomatis dan permanen ke dalam sel.
Ctrl + Shift + $ (Angka 4) Currency Format: Mengubah format angka menjadi format mata uang secara instan (misal: menambahkan Rp atau $).
Ctrl + Shift + % (Angka 5) Percentage Format: Mengubah tampilan angka desimal menjadi bentuk persentase secara instan.
Ctrl + Shift + ! (Angka 1) Number Format: Mengatur angka dengan format dua angka di belakang koma desimal beserta tanda pemisah ribuan.
Ctrl + 1 Format Cells Menu: Membuka kotak dialog utama Format Cells untuk mengatur font, border, alignment, dan jenis data secara custom.
Shift + Spasi Select Row: Menyeleksi atau memblok satu baris penuh secara horizontal dari ujung kiri ke kanan.
Ctrl + Spasi Select Column: Menyeleksi atau memblok satu kolom penuh secara vertikal dari ujung atas ke bawah.
Ctrl + + (Plus) Insert Row/Column: Menambahkan baris atau kolom baru yang kosong (blok baris/kolom terlebih dahulu agar lebih cepat).
Ctrl + - (Minus) Delete Row/Column: Menghapus baris atau kolom yang dipilih secara instan.
Ctrl + 9 Hide Row: Menyembunyikan baris data yang sedang dipilih agar tidak terlihat di layar kerja.
Ctrl + 0 (Nol) Hide Column: Menyembunyikan kolom data yang sedang dipilih.
Ctrl + T Create Table: Mengubah sekumpulan data mentah yang diblok menjadi format tabel resmi Excel lengkap dengan fitur sorting dan filter otomatis yang keren.
F11 Create Chart: Membuat grafik visual (chart) instan di lembar kerja terpisah berdasarkan data yang sedang diblok.
Alt + ↓ Drop-Down List: Menampilkan daftar pilihan drop-down dari data unik yang pernah diinput pada kolom tersebut di atasnya.
