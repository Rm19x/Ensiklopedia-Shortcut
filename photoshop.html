Daftar Lengkap Shortcut Adobe Photoshop (Windows)

Kombinasi Tombol Fungsi / Deskripsi Nyata
Ctrl + Shift + N New Layer: Membuat layer baru yang kosong di atas layer yang sedang aktif aktif saat ini.
Ctrl + J Layer via Copy: Menduplikat/menggandakan layer aktif, atau menduplikat area seleksi tertentu ke dalam layer baru secara instan.
Ctrl + G Group Layers: Memasukkan layer-layer yang sedang dipilih ke dalam satu folder grup agar panel workspace rapi.
Ctrl + E Merge Layers: Menggabungkan layer yang dipilih dengan layer tepat di bawahnya menjadi satu raster gambar tunggal.
Ctrl + Shift + Alt + E Stamp Visible: Membuat satu layer cuplikan baru di posisi paling atas yang berisi gabungan dari seluruh layer kasat mata tanpa merusak struktur layer asli di bawahnya.
Ctrl + [ / ] Move Layer Order: Menurunkan [ atau menaikkan ] posisi urutan layer aktif di panel layers.
Ctrl + T Free Transform: Memunculkan kotak kontrol untuk mengubah ukuran (*resize*), memutar (*rotate*), atau memiringkan objek gambar.
Ctrl + D Deselect: Menghilangkan garis putus-putus seleksi (marching ants) yang sedang aktif di layar kerja.
Ctrl + Shift + I Inverse Selection: Membalikkan area seleksi (memilih area yang sebelumnya berada di luar seleksi).
Ctrl + Alt + R Select and Mask: Membuka panel isolasi tingkat lanjut untuk merapikan potongan tepi seleksi rambut atau bulu halus secara presisi.
V Move Tool: Mengaktifkan alat penggeser objek atau gambar di kanvas kerja.
M Marquee Tool: Mengaktifkan alat pembuat seleksi berbentuk kotak atau lingkaran sempurna.
L Lasso Tool: Mengaktifkan alat seleksi bebas menggunakan gerakan mouse tanganmu.
W Magic Wand / Quick Selection: Mengaktifkan pembuat seleksi otomatis berbasis kesamaan warna atau deteksi objek pintar.
B Brush Tool: Mengaktifkan alat kuas lukis digital utama.
E Eraser Tool: Mengaktifkan alat penghapus objek gambar pada layer aktif.
P Pen Tool: Mengaktifkan alat pembuat garis vektor atau *path* pemotongan gambar dengan akurasi tertinggi.
[ / ] (Saat Brush Aktif) Change Brush Size: Mengecilkan [ atau memperbesar ] diameter ukuran brush secara instan tanpa masuk ke menu atas.
Shift + [ / ] Change Brush Hardness: Mengurangi [ (menjadi lembut/soft) atau menambah ] (menjadi tegas/hard) efek tepi goresan brush.
X Switch Colors: Menukar posisi warna utama (Foreground) dengan warna latar belakang (Background) pada kotak palet warna.
D Default Colors: Mengembalikan setelan palet warna menjadi hitam murni (Foreground) dan putih murni (Background) secara standar awal.
Spasi + Klik & Tarik Hand Tool (Navigasi): Mengubah kursor menjadi tangan untuk menggeser posisi layar kanvas saat kamu sedang melakukan zoom-in mendalam.
Ctrl + L Levels: Membuka panel grafik histogram untuk mengatur kecerahan bayangan (*shadow*) dan highlights gambar.
Ctrl + M Curves: Membuka kurva tonal warna tingkat lanjut untuk mengatur kontras dan pencahayaan gambar secara dinamis.
Ctrl + U Hue / Saturation: Membuka pengaturan untuk mengubah warna dasar, kepekatan keaslian warna (*saturation*), dan tingkat kegelapan objek.
Ctrl + Shift + A Camera Raw Filter: Membuka panel editing eksternal super lengkap untuk koreksi warna tingkat fotografer profesional.
