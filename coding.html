Kombinasi Tombol Fungsi / Deskripsi Nyata
Alt + ↑ / ↓ Move Line: Memindahkan satu baris kode (atau blok kode yang diseleksi) ke atas atau ke bawah tanpa perlu cut-paste.
Shift + Alt + ↑ / ↓ Copy Line: Menggandakan (duplikat) baris kode yang sedang aktif langsung ke atas atau ke bawahnya.
Ctrl + Shift + K Delete Line: Menghapus satu baris tempat kursor berada secara instan.
Ctrl + Enter Insert Line Below: Membuat baris baru kosong tepat di bawah baris aktif, tanpa peduli posisi kursor sedang di tengah atau ujung kalimat.
Ctrl + Shift + Enter Insert Line Above: Membuat baris baru kosong tepat di atas baris aktif saat ini.
Ctrl + / Toggle Line Comment: Mengubah baris kode menjadi komentar (comment) atau sebaliknya secara otomatis sesuai bahasa pemrograman yang dipakai.
Shift + Alt + A Toggle Block Comment: Membungkus kode yang diseleksi menjadi blok komentar multi-baris (misal: /* kode */).
Alt + Klik Kiri Insert Cursor: Menaruh banyak kursor di berbagai baris berbeda sekaligus untuk mengetik secara massal.
Ctrl + Alt + ↑ / ↓ Column Selection: Menambahkan kursor lurus ke atas atau ke bawah secara vertikal untuk mengedit kode berbentuk tabel/list.
Ctrl + D Select Next Match: Menyeleksi kata yang sama berikutnya satu per satu. Sangat cepat untuk mengubah nama variabel yang berulang.
Ctrl + Shift + L Select All Matches: Menyeleksi SELURUH kata yang sama di dalam satu file aktif sekaligus untuk diubah massal.
Shift + Alt + → Expand Selection: Memperluas seleksi teks secara pintar berbasis struktur kode (karakter → kata → string → fungsi).
Shift + Alt + ← Shrink Selection: Mempersempit kembali cakupan seleksi pintar yang telah diperluas.
Ctrl + P Quick Open File: Mencari dan membuka file apapun di dalam folder proyek secara instan cukup dengan mengetik namanya.
Ctrl + T Show All Symbols: Mencari fungsi, class, atau variabel spesifik di seluruh file proyek.
Ctrl + G Go to Line: Melompat langsung ke nomor baris tertentu (ketik nomor baris setelah tanda titik dua :).
Ctrl + F Find in File: Membuka kolom pencarian teks di dalam file yang sedang aktif.
Ctrl + H Replace in File: Menemukan teks dan menggantinya dengan teks baru dalam file aktif.
Ctrl + Shift + F Global Search: Mencari teks atau kata kunci spesifik di SELURUH file yang ada di dalam folder proyek.
Ctrl + B Toggle Sidebar: Menyembunyikan atau menampilkan sidebar kiri (File Explorer) agar area menulis kode menjadi lebih luas.
Ctrl + \ (Backslash) Split Editor: Membagi layar editor kode menjadi dua atau tiga bagian secara vertikal untuk membandingkan kode side-by-side.
Ctrl + 1 / 2 / 3 Focus Group: Berpindah fokus keyboard ke jendela editor grup pertama, kedua, atau ketiga hasil dari split screen.
Ctrl + W Close Editor: Menutup tab file yang sedang aktif digunakan saat ini.
Ctrl + ` (Backtick) Toggle Integrated Terminal: Membuka atau menutup panel terminal internal bawaan VS Code di bagian bawah tanpa merusak layout.
Ctrl + Shift + ` Create New Terminal: Membuat *instance* atau tab terminal baru yang bersih.
Shift + Alt + F Format Document: Merapikan struktur penulisan kode (indentasi, spasi, semi-kolon) secara otomatis sesuai standar linter (Prettier/Beautify).
Ctrl + Space Trigger Suggestion: Memunculkan paksa saran pelengkap otomatis (IntelliSense/Autocompletion) jika sempat tertutup.
F12 Go to Definition: Melompat langsung ke asal-usul fungsi atau variabel dideklarasikan untuk memeriksa strukturnya.
Alt + Z Toggle Word Wrap: Mengaktifkan fitur bungkus teks agar kode yang terlalu panjang ke kanan otomatis turun ke bawah tanpa perlu horizontal scrolling.
