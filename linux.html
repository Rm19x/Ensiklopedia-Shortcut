Kombinasi Tombol Fungsi / Deskripsi Nyata
Ctrl + Alt + T Open Terminal: Shortcut paling sakral di Linux untuk membuka jendela Terminal baru secara instan.
Super (Tombol Win) Activities Overview: Membuka menu aplikasi, kolom pencarian sistem, dan melihat semua jendela yang sedang aktif.
Alt + F2 Run Command Box: Membuka kotak dialog cepat untuk menjalankan perintah aplikasi atau merestart window manager.
Super + L Lock Screen: Mengunci layar sesi user Linux secara instan.
Ctrl + Alt + Del Log Out: Membuka dialog konfirmasi untuk keluar dari sesi pengguna aktif atau mematikan sistem.
Super + D atau Ctrl + Alt + D Hide All Windows: Menyembunyikan seluruh jendela yang terbuka untuk langsung menampilkan desktop bersih.
Super + Page Up / Page Down Switch Workspace: Berpindah ruang kerja (workspace/virtual desktop) ke atas atau ke bawah.
Super + Shift + Page Up / Page Down Move Window to Workspace: Memindahkan jendela aplikasi yang sedang aktif ke workspace lain secara instan.
Ctrl + A Move to Start: Memindahkan kursor teks langsung ke awal baris perintah di Terminal tanpa menekan panah kiri berulang kali.
Ctrl + E Move to End: Memindahkan kursor teks langsung ke ujung/akhir baris perintah di Terminal.
Ctrl + U Clear Line Left: Menghapus seluruh teks perintah dari posisi kursor saat ini ke arah kiri (awal baris).
Ctrl + K Clear Line Right: Menghapus seluruh teks perintah dari posisi kursor saat ini ke arah kanan (akhir baris).
Ctrl + W Delete Word Left: Menghapus satu kata tepat di sebelah kiri kursor di Terminal.
Ctrl + Y Paste/Yank: Memunculkan kembali (paste) teks terakhir yang baru saja dihapus menggunakan Ctrl+U atau Ctrl+K.
Ctrl + ← / → atau Alt + B / F Jump Word: Melompati kursor per satu kata penuh ke arah kiri atau kanan.
Ctrl + C Kill Process: Menghentikan secara paksa (*SIGINT*) proses atau script yang sedang berjalan di Terminal.
Ctrl + Z Suspend Process: Menghentikan sementara (*SIGTSTP*) proses yang berjalan dan memindahkannya ke latar belakang (*background*).
Ctrl + D Exit Session: Menutup sesi Terminal aktif atau keluar dari koneksi SSH saat ini (Sama dengan mengetik 'exit').
Ctrl + R Reverse Search History: Fitur ajaib untuk mencari perintah-perintah lama yang pernah kamu ketik di Terminal cukup dengan mengetik kata kuncinya.
Ctrl + G Quit Search: Keluar dari mode pencarian sejarah Ctrl+R tanpa mengeksekusi perintah apapun.
Ctrl + L Clear Screen: Membersihkan layar Terminal dari teks-teks sebelumnya tanpa menghapus riwayat perintah (Sama dengan mengetik 'clear').
Ctrl + S Freeze Terminal: Menghentikan output layar Terminal sementara (terlihat seperti hang, berguna untuk membaca log cepat).
Ctrl + Q Unfreeze Terminal: Mengembalikan status Terminal yang beku akibat perintah Ctrl+S agar bisa mengetik kembali.
Ctrl + Shift + C Copy in Terminal: Menyalin (copy) teks yang diblok di dalam Terminal (Ctrl+C biasa tidak bekerja di sini karena fungsinya untuk kill proses).
Ctrl + Shift + V Paste in Terminal: Menempelkan (paste) teks ke dalam Terminal.
Print Screen Take Screenshot: Mengambil tangkapan seluruh layar sistem.
Alt + Print Screen Window Screenshot: Mengambil tangkapan layar khusus pada jendela aplikasi yang sedang aktif saja.
Shift + Print Screen Area Screenshot: Memilih area spesifik di layar menggunakan kursor untuk difoto.
Ctrl + Alt + F1 sampai F6 Switch to TTY (Virtual Console): Keluar dari mode grafis/desktop langsung menuju konsol teks murni (Sangat berguna jika desktop Linux-mu freeze/crash total untuk memperbaiki sistem lewat CLI).
Ctrl + Alt + F7 atau F1 Return to GUI: Kembali ke lingkungan Desktop Grafis semula dari mode TTY.
