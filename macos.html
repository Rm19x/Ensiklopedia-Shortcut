Daftar Lengkap Shortcut macOS

Kombinasi Tombol Fungsi / Deskripsi Nyata
Cmd ⌘ + Spasi Spotlight Search: Membuka kolom pencarian sakral macOS untuk mencari file, meluncurkan aplikasi, hingga menghitung matematika instan.
Cmd ⌘ + Tab Switch Apps: Berpindah ke aplikasi aktif berikutnya yang sedang berjalan di latar belakang.
Cmd ⌘ + Q Quit App: Menutup aplikasi yang sedang aktif secara total (bukan cuma menyembunyikan jendela seperti tombol silang merah).
Cmd ⌘ + W Close Window: Menutup jendela aplikasi yang sedang aktif tanpa mematikan proses aplikasinya.
Cmd ⌘ + M Minimize Window: Mengecilkan jendela aktif saat ini ke Dock di bagian bawah layar.
Cmd ⌘ + Option ⌥ + Esc Force Quit: Membuka jendela darurat untuk mematikan paksa aplikasi yang mengalami crash atau *freeze* (Sama seperti Task Manager di Windows).
Cmd ⌘ + Shift ⇧ + 3 Full Screenshot: Mengambil foto seluruh tampilan layar macOS dan menyimpannya langsung ke Desktop sebagai file gambar.
Cmd ⌘ + Shift ⇧ + 4 Area Screenshot: Mengubah kursor menjadi target bidik untuk mengambil tangkapan layar di area spesifik pilihanmu.
Cmd ⌘ + Shift ⇧ + 4 lalu Spasi Window Screenshot: Mengambil foto khusus satu jendela aplikasi tertentu secara bersih dengan efek bayangan khas macOS yang rapi.
Cmd ⌘ + Shift ⇧ + 5 Screen Recording & Utility: Membuka panel antarmuka lengkap untuk melakukan tangkapan layar maupun merekam video layar mac.
Cmd ⌘ + Option ⌥ + Spasi Open Finder Search: Membuka jendela Finder baru dalam posisi siap mencari berkas secara mendalam.
Spasi (Setelah memilih file) Quick Look: Fitur ajaib untuk melihat pratinjau (preview) isi file dokumen, foto, atau video secara instan tanpa perlu membuka aplikasi pendukungnya.
Cmd ⌘ + Delete ⌫ Move to Trash: Membuang berkas atau folder yang dipilih langsung ke dalam Trash (Keranjang Sampah).
Cmd ⌘ + Shift ⇧ + Delete ⌫ Empty Trash: Mengosongkan keranjang sampah secara permanen tanpa perlu konfirmasi klik kanan pada Dock.
Return / Enter (Setelah memilih file) Rename File: Cara cepat untuk mengubah nama berkas atau folder terpilih (Bukan untuk mengeksekusi file).
Cmd ⌘ + I Get Info: Membuka panel detail properti berkas (kapasitas size, ekstensi berkas, hak akses enkripsi).
Cmd ⌘ + Shift ⇧ + N New Folder: Membuat direktori folder baru di dalam Finder secara instan.
Cmd ⌘ + C Copy: Menyalin teks atau file yang dipilih ke dalam papan klip (*clipboard*).
Cmd ⌘ + V Paste: Menempelkan hasil salinan teks atau berkas ke tujuan baru.
Cmd ⌘ + Option ⌥ + V Move File (Cut-Paste Mac Style): Solusi memindahkan file di Mac. Lakukan Copy biasa dengan Cmd+C, lalu tekan shortcut ini di folder tujuan untuk memindahkannya (Cut).
Cmd ⌘ + Ctrl ⌃ + Spasi Character Viewer (Emoji): Memunculkan jendela pop-up karakter khusus untuk mengetik emoji dan simbol universal di macOS.
Cmd ⌘ + Z Undo: Membatalkan aksi terakhir yang baru saja kamu lakukan.
Cmd ⌘ + Shift ⇧ + Z Redo: Mengulang kembali aksi yang dibatalkan oleh perintah Undo sebelumnya.
Option ⌥ + ← / → Jump Word: Menggerakkan kursor teks melompati satu kata penuh ke arah kiri atau kanan dengan presisi.
Cmd ⌘ + ← / → Jump to End/Start Line: Melompat langsung ke awal kalimat paling kiri atau ke ujung kalimat paling kanan di baris aktif.
