Daftar Lengkap Shortcut Windows

Kombinasi Tombol Fungsi / Deskripsi Nyata
Win + E File Explorer: Membuka jendela File Explorer secara instan dari mana saja.
Win + D Show/Hide Desktop: Meminimalkan semua jendela yang terbuka secara instan untuk melihat desktop, tekan lagi untuk mengembalikan.
Win + L Lock Screen: Mengunci komputer seketika saat kamu harus meninggalkan meja kerja.
Win + I Settings: Membuka menu Pengaturan/Settings utama Windows.
Win + X Quick Link Menu: Membuka menu akses cepat di pojok kiri bawah (Device Manager, Terminal, Disk Management, dll).
Win + R Run Dialog Box: Membuka kotak dialog 'Run' untuk mengeksekusi perintah cepat seperti cmd, regedit, atau msconfig.
Ctrl + Shift + Esc Task Manager: Cara tercepat membuka Task Manager secara langsung tanpa melalui layar interupsi Ctrl+Alt+Del.
Alt + F4 Close / Shutdown: Menutup aplikasi yang sedang aktif, atau membuka menu Shutdown jika ditekan di posisi desktop.
Alt + Tab Switch Apps: Berpindah antar aplikasi atau jendela yang sedang terbuka secara cepat.
Win + Tab Task View: Menampilkan semua jendela aktif dalam bentuk grid serta akses ke Virtual Desktop.
Win + ← / → Snap Left/Right: Menempelkan jendela aktif ke setengah layar bagian kiri atau kanan (Sangat berguna untuk split-screen).
Win + ↑ / ↓ Maximize/Minimize Jendela: Memaksimalkan ukuran jendela hingga satu layar penuh, atau mengecilkan/meminimalkannya ke taskbar.
Win + Home Minimize Background Windows: Meminimalkan seluruh jendela di latar belakang kecuali jendela aktif yang sedang kamu gunakan saat ini.
Win + Ctrl + D Create Virtual Desktop: Membuat ruang kerja desktop virtual baru yang bersih secara instan.
Win + Ctrl + ← / → Switch Virtual Desktop: Berpindah dengan cepat di antara desktop virtual yang telah kamu buat.
Win + Ctrl + F4 Close Virtual Desktop: Menutup desktop virtual yang sedang aktif digunakan saat ini.
Win + V Clipboard History (Sangat Penting): Membuka riwayat teks atau gambar yang dicopy sebelumnya, bukan cuma data terakhir (Harus diaktifkan sekali saat pertama dipakai).
Win + Shift + S Advanced Screenshot (Snip & Sketch): Mengambil tangkapan layar area spesifik (kotak, bebas, atau satu layar) secara instan untuk langsung di-paste.
Win + . (Titik) atau ; Emoji & GIF Panel: Memunculkan jendela pop-up berisi emoji, GIF, dan simbol matematika/bahasa rahasia secara universal.
Win + G Xbox Game Bar: Membuka panel perekam layar bawaan Windows (Screen Recorder), kontrol audio, dan performa hardware.
Win + Alt + R Instant Screen Recording: Langsung memulai atau menghentikan perekaman layar video tanpa harus membuka Game Bar terlebih dahulu.
Ctrl + Shift + N New Folder: Membuat folder baru secara instan di dalam folder aktif pada File Explorer.
Alt + Enter File Properties: Menampilkan jendela properti (ukuran file, tipe, izin) dari file atau folder yang sedang dipilih.
Shift + Delete Permanent Delete: Menghapus file secara permanen tanpa membuangnya ke Recycle Bin terlebih dahulu (Gunakan dengan hati-hati!).
F2 Rename File: Mengubah nama file atau folder yang sedang dipilih secara instan tanpa klik kanan.
F5 atau Ctrl + R Refresh: Memperbarui tampilan jendela aktif atau halaman File Explorer.
Ctrl + Z Undo: Membatalkan tindakan terakhir (Berlaku juga untuk pembatalan penghapusan file di File Explorer).
Ctrl + Y Redo: Mengulang kembali tindakan yang baru saja dibatalkan oleh perintah Undo.
Ctrl + A Select All: Memilih seluruh elemen, file dalam folder, atau teks di dalam dokumen secara instan.
Win + P Project Presentation: Mengatur proyeksi layar monitor saat menggunakan proyektor atau monitor ganda (Duplicate, Extend, Second Screen Only).
Win + K Cast Panel: Membuka menu koneksi cepat untuk menyambungkan layar ke perangkat wireless eksternal (Miracast/Smart TV).
Win + A Action Center / Quick Settings: Membuka panel pengaturan cepat (Wi-Fi, Bluetooth, Kecerahan Layar, Airplane Mode).
Win + N Notification & Calendar: Membuka pusat notifikasi sistem beserta kalender bulanan (Khusus Windows 11).
Win + Shift + Ctrl + B Reset Graphics Driver (Penyelamat PC): Merestart driver kartu grafis/VGA secara instan untuk mengatasi layar freeze, blank hitam, atau lag parah tanpa restart komputer.
