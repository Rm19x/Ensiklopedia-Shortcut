Kombinasi Tombol Fungsi / Deskripsi Nyata
Ctrl + A Jump to Start: Memindahkan kursor teks langsung ke awal baris perintah aktif tanpa perlu menekan panah kiri berulang-ulang.
Ctrl + E Jump to End: Memindahkan kursor teks langsung ke ujung paling akhir dari baris perintah.
Alt + B atau Ctrl + ← Back One Word: Memindahkan kursor mundur sejauh satu kata penuh ke arah kiri.
Alt + F atau Ctrl + → Forward One Word: Memindahkan kursor maju sejauh satu kata penuh ke arah kanan.
Ctrl + U Cut Left to Start: Menghapus (cut) semua karakter dari posisi kursor saat ini ke arah kiri sampai ke awal baris.
Ctrl + K Cut Right to End: Menghapus (cut) semua karakter dari posisi kursor saat ini ke arah kanan sampai ke akhir baris.
Ctrl + W Delete Word Left: Menghapus satu kata tepat di sebelah kiri posisi kursor.
Alt + D Delete Word Right: Menghapus satu kata tepat di sebelah kanan posisi kursor.
Ctrl + Y Paste/Yank: Menempelkan kembali (paste) elemen teks terakhir yang baru saja dihapus menggunakan perintah Ctrl+U, Ctrl+K, atau Ctrl+W.
Ctrl + _ (Underscore) Undo CLI Action: Membatalkan ketikan atau penghapusan teks terakhir yang kamu lakukan di dalam terminal.
Ctrl + R Reverse Intelligent Search: Membuka mode pencarian riwayat teks pintar untuk memanggil kembali perintah panjang masa lalu hanya dengan mengetik serpihan kata kuncinya.
Ctrl + G Exit History Mode: Keluar dari mode pencarian Ctrl+R tanpa mengeksekusi perintah apapun dan mengembalikan ketikan semula.
Ctrl + P atau ↑ Previous Command: Melihat dan memunculkan kembali satu perintah terakhir di baris riwayat atas.
Ctrl + N atau ↓ Next Command: Berpindah ke baris perintah berikutnya yang lebih baru di dalam riwayat chat terminal.
Ctrl + L Clear Screen Display: Membersihkan seluruh tampilan layar terminal dari tumpukan teks log tanpa menghapus riwayat perintah yang tersimpan (Sama seperti mengetik clear atau cls).
Ctrl + S Freeze Terminal Output: Menghentikan sementara aliran data output pada layar (sangat berguna untuk membaca log server yang berjalan terlalu cepat).
Ctrl + Q Unfreeze Terminal Output: Melepaskan status beku akibat perintah Ctrl+S agar terminal bisa beroperasi normal kembali.
Tab Autocompletion: Fitur otomatisasi paling penting untuk melengkapi nama perintah, folder, atau file secara cerdas tanpa perlu mengetik penuh.
Ctrl + C Terminate Process: Mengirimkan sinyal interrupt (SIGINT) untuk menghentikan secara paksa script atau proses aplikasi yang sedang berjalan di terminal.
Ctrl + Z Suspend Process: Menghentikan sementara proses aktif dan memindahkannya ke latar belakang (background), ketik fg untuk mengembalikannya ke depan.
Ctrl + D End Terminal Session: Menutup aplikasi terminal saat ini atau memutus jaringan koneksi remote SSH secara bersih (Sama dengan perintah exit).
